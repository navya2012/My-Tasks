DESCENDANT SELECTORS

This selector is used to select all the child elements (or) direct child elements of the specified tag.

DIRECT CHILDREN SELECTORS

This child selector selects all elements that are specified in that parent element.

ADJACENT SIBILING SELECTORS

The Adjacent sibling selector is used to select the element that is next to the specified selector tag. selects only one tag that is just next to the specified tag.

GENERAL SIBILING SELECTORS

The general sibling selector selects all elements that are next siblings of a specified element.
