GRID LAYOUT

A grid layout consists of a parent element, with one or more child elements.
Grid-based layout system, with rows and columns, makes it easier to design web pages.
The vertical lines of grid items are called columns.
The horizontal lines of grid items are called rows.
The spaces between each column are called column-gaps.
The spaces between each row are called row-gaps
The gap is a shorthand property for the row-gap and the column-gap properties
The gap can also be used to set both the row gap and the column gap in one value
The lines between columns are called column lines.
